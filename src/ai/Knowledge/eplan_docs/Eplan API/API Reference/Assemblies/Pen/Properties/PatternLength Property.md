# PatternLength Property

**Source URL:** https://www.eplan.help/en-us/Infoportal/Content/api/2026/Eplan.EplApi.DataModelu~Eplan.EplApi.DataModel.Graphics.Pen~PatternLength.html

Pattern length. The pattern will be stretched, so that the whole pattern fits to the length of PatternLength. This is like in EPLAN 5. Please use "-16002" as "from layer" value.
