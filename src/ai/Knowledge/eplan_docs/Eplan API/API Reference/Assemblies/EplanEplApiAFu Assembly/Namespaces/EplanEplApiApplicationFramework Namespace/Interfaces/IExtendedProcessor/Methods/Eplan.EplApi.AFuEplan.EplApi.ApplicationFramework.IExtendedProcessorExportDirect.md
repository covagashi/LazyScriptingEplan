Exports to a special file. All Parameter are in the context.

Syntax

* [C#](#i-syntax-CS)
* [C++/CLI](#i-syntax-CPP2005)

```
```
bool ExportDirect( 
   string strOutputFile,
   Context oContext
)
```
```

```
```
bool ExportDirect( 
   String^ strOutputFile,
   Context^ oContext
)
```
```

#### Parameters

*strOutputFile*
:   Output file

*oContext*
:   Context with parameters

#### Return Value

Returns true if successful.
