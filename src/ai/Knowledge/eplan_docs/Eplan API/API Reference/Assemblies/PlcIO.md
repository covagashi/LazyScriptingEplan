# PlcIO

**Source URL:** https://www.eplan.help/en-us/Infoportal/Content/api/2026/Eplan.EplApi.DataModelu~Eplan.EplApi.DataModel.PlcIO.html

---

Class representing a PlcIOEntry object. A PlcIOEntry object is an entity that holds some data (hardware address, symbolic address, data type, CPU id) that can be assigned (booked) to a PLC IO terminal. Free PlcIOEntries are created in P8 project by importing an assignment list. Such free PlcIOEntries may then be booked onto existing PLC terminals. When PLCIOEntry is booked onto a PLC terminal, the data is transfered to the terminal and the entry is deleted from project.
