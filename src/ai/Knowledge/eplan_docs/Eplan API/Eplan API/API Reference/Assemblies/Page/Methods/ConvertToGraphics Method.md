# ConvertToGraphics Method

**Source URL:** https://www.eplan.help/en-us/Infoportal/Content/api/2026/Eplan.EplApi.DataModelu~Eplan.EplApi.DataModel.Page~ConvertToGraphics.html

Converts all symbols and connections on the page into graphical instances. Graphical instances of a given symbol are automatically grouped.

- [C++/CLI](#i-syntax-CPP2005)

public Placement[] ConvertToGraphics()
```
```

```
```

```
```

#### Return Value

An array of all symbols and auto connecting lines converted into graphical instances.
